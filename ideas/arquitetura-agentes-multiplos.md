# Arquitetura de Agentes Múltiplos

## Origem
- File: `knowledge/02-technical-decisions.md`
- File: `notes/03-iteration-log.md`

## Descrição
Explorar arquitetura de múltiplos agentes especializados trabalhando em conjunto, inspirado na arquitetura do Neo Brain.

## Contexto
- Neo Brain opera como sistema de gerenciamento de conhecimento
- Conceito de "hot swap" (mudança de direção mid-course) mostra necessidade de agentes adaptativos
- Sistema de iteração permite aprendizado contínuo

## Pensamentos
- Agent especializado para busca de informações
- Agent especializado para execução de tarefas
- Agent especializado para síntese e apresentação
- Coordenador central para orquestrar fluxos
